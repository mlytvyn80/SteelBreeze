void SBOceanTideLd::ARG_IERS ( const SBTime &  Time) [protected]

COMPUTES THE ANGULAR ARGUMENT WHICH DEPENDS ON TIME FOR 11 TIDAL ARGUMENT CALCULATIONS. This strange subroutine has been printed in the IERS Technical Note 21, IERS Conventions (1996). It is assumed that it calculates the "(omega_j*t + hi_j)" part of the equation (4), p.53 (however, it is not stated explicitly).
